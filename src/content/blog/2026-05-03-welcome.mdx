このサイトのブログを開設しました。研究の進捗、論文紹介、ケモインフォマティクスや機械学習関連のメモなどを書いていく予定です。

## 技術スタック

このブログは [Astro](https://astro.build) + MDX + Tailwind CSS で動いています。記事は Markdown 記法で書け、必要に応じて Astro/React コンポーネントを埋め込むこともできます。

例えば、こんな感じでコードブロックも使えます。

```python
from rdkit import Chem
mol = Chem.MolFromSmiles("CCO")
print(mol.GetNumAtoms())
```
